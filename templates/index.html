        Data Preprocessing & Pipeline

        How we cleaned the raw diabetes.csv dataset before training the model.

        • Feature Selection: Filtered dataset down to the 4 most critical clinical metrics (Glucose, Blood Pressure, BMI, Age).
        • Handling Missing Data: In this dataset, a value of 0 for Glucose, Blood Pressure, or BMI is biologically impossible and represents missing data. We replaced these 0s with NaN.
        • Median Imputation: We filled all missing NaN values with the median value of their respective columns to prevent data loss.
        • Feature Scaling: We applied StandardScaler to normalize the data, ensuring features with larger ranges (like Glucose) don't unfairly dominate smaller features (like BMI) during model training.
